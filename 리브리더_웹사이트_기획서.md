## 1. 사이트맵 & 메뉴

> **구조 벤치마킹 원칙**: 바이버(viver.co.kr)의 정보구조·레이아웃 그리드·페이지 흐름(홈→카테고리→리스트→상세→마이페이지 등 구성 방식, 넓은 여백의 에디토리얼 그리드)을 골격으로 그대로 채택. 단, 실제 사진·일러스트·카피·고유 UI 디테일은 바이버 것을 복제하지 않고 TEAMCCR 고유 콘텐츠(11장 브랜드 DB, 12장 강사 프로필, 인증마크 등)로 채움.

### GNB (바이버와 슬롯 개수·역할 일치)
```
[TEAMCCR 로고]     [언어 🌐] [검색 🔍] [알림 🔔]
                홈 | 검색 | TEAMCCR | 매거진 | 판매 | 마이페이지
```
- 1행: 로고 + 우측 유틸리티 아이콘(언어전환·퀵서치·알림) — 바이버와 동일
- 2행: 메인 내비게이션 6개, **마이페이지는 아이콘이 아니라 메뉴 마지막 항목**(바이버와 동일하게 수정)
- **판매**는 바이버처럼 브라우징(홈/검색)과 분리된 독립 최상위 메뉴 — 매입형/위탁형 신청 진입점(직거래 없음, 모든 매물은 TEAMCCR 검수 후 게시)
- **TEAMCCR**은 바이버의 "VIVER ONE" 슬롯에 대응 — 강사진·스케줄 캘린더를 최상위 노출로 유지(확정)
- **오버홀·정비**와 **교육**은 바이버에 대응 슬롯이 없음(바이버의 "수리/진단"도 GNB가 아니라 홈 퀵아이콘) → 최상위 메뉴에서 빼고 **홈 퀵아이콘 + 마이페이지 하위**로 이동
- 고객센터·정책 페이지들은 바이버처럼 GNB 노출 없이 **마이페이지 하위 메뉴**로 이동

### 홈 화면 구성 (바이버 벤치마킹)
1. **히어로 프로모션 배너**: 캐러셀 형태(예: 1/N 인디케이터), 이벤트·프로모션 노출용
2. **퀵아이콘 숏컷 행**: 바이버의 "할인&적립·쇼룸예약·모먼트·AI렌즈·매거진·수리/진단"에 대응 —
   - **검수(진단) 신청** (바이버 "수리/진단"에 대응 — 판매 목적 없이 단독으로도 신청 가능)
   - 오버홀 신청 (검수와 별개, 선택적 부가서비스)
   - 센터 방문예약 (바이버 "쇼룸예약"에 대응)
   - 인증서 조회 (바이버 "AI렌즈"에 대응 — 스캔으로 확인한다는 인터랙션 패턴 동일)
   - 교육 신청
   - 매거진
3. **브랜드 바로가기 행**: "전체보기" + 취급 브랜드 로고 가로 스크롤(11장 브랜드 DB: AP Diving·Divesoft·JJ-CCR·KISS·Halcyon·rEvo 등) → 클릭 시 해당 브랜드 매물로 필터링
4. **신규 등록 하이라이트 섹션**: "이번 주 등록된 신규 매물"을 **좌우 슬라이드 캐러셀**로 노출(그리드 아님, 가로 스크롤), "금주 등록" 뱃지 + 더보기 링크 — 홈 화면의 상품 목록은 모두 이 방식으로 통일. 전체 목록을 보여주는 "검색" 페이지는 그리드 유지

### 검색 (구매 브라우징 통합 — 구 "중고거래 구매하기" + "새상품")

**브랜드 슬라이더 + 즉시 필터** (바이버와 동일 UX): 상단에 "전체" + 브랜드 로고가 가로 슬라이드로 노출, 브랜드 클릭 시 페이지 이동 없이 바로 아래 목록이 해당 브랜드로 필터링됨(11장 브랜드 DB 기반).

**탭 구조**: "상품 N" / "모델 N" 2개 탭
- 상품 탭: 실제 등록된 개별 매물 수
- 모델 탭: 브랜드 DB 기준 모델 종류 수(예: 11장 DB의 고유 모델 개수) — 클릭 시 모델별 대표 이미지·시세 요약 페이지로 이동(바이버의 "모델 정보 & 시세" 개념)

**필터 칩(바이버 "VIVER Ready/미착용/쇼룸/CARE" 대응)**: 즉시구매가능 · 신품급 · 센터방문가능 · CARE(TEAMCCR 케어 멤버십 대상) — 정렬(최신순 등) 드롭다운 동반

**리스트 카드**: "오늘 등록" 유형 뱃지, 대표 이미지, 브랜드/모델명, 스펙 요약(마운트타입·제어방식·사용시간), 가격, 신뢰 뱃지(인증여부·센터 위치), 조회수

**검색 필터 모달** (바이버는 시계 특성 필터라 리브리더 특성으로 전면 교체):
| 바이버 원본 | TEAMCCR 대체 |
|---|---|
| 사이즈(mm) 슬라이더+프리셋 | **마운트타입**(BM/CM/SM) + **제어방식**(mCCR/eCCR/Hybrid/SCR) 프리셋 버튼 |
| 가격 슬라이더(0원~1억원 이상) | 가격 슬라이더 동일 구조 유지 |
| 스탬핑(생산연도) 슬라이더+프리셋 | 구매/제조연도 슬라이더+프리셋 유지(연식 중요도 유사) |
| 보증서 있음/상관없음 | **상태등급 필터**(S/A/B/C) — 모든 매물이 TEAMCCR 검수를 거치므로 인증여부 자체는 필터 불필요, 등급으로 구분 |
| — | (신규) 누적 다이빙 횟수/사용시간 범위, 오버홀 이력 유무 |

- 새상품(브랜드사 공식 딜러 카탈로그, 사전예약/입고알림, 정품등록 안내)도 이 통합 리스트 안에서 필터로 구분

### 상품 상세페이지 (바이버 상세 화면 구조 그대로 채택)

1. **이미지 캐러셀**: 1/N 인디케이터 + **360° 뷰 옵션**(리브리더도 유닛 전체 형상 확인에 유용)
2. **신뢰 뱃지 행**: 즉시구매가능 · 센터 위치 · 상태등급(신품급 등)
3. **타이틀**: 브랜드 / 모델명 / 스펙 한 줄 요약(마운트·제어방식·사용시간)
4. **스펙 아이콘 행**: 검수 인증서 유무(전건 발급) · 오버홀 이력 유무 · 구성품 포함 여부(카운터렁/캐니스터/호스 등) · 특이사항
5. **가격 + 구매하기(sticky 버튼)**
6. **가입 유도 배너**: "구매 시 함께 가입 가능한 서비스" → **TEAMCCR CARE**(무상 수리 혜택, 정기점검 등 멤버십) 업셀 — 부록 9의 케어 멤버십과 연결되는 실제 노출 지점
7. **배송정보**: 빠른 배송(1~2일 배송/센터 픽업) · **바로 판매**(배송 없이 구매 즉시 재판매 — 매입형 재고 회전에 활용 가능한 옵션으로 검토)
8. **판매정보**: 매입형/위탁형 여부 명시
9. **상품번호**: 내부 관리번호(시리얼 매칭)
10. **센터 방문 CTA**: "센터에서 실물 확인 가능한 상품입니다 → 방문 예약하기"
11. **모델 정보 & 시세 링크**: 해당 모델의 DB 페이지로 연결(11장)
12. **TEAMCCR 인증 결과** (바이버 "바이버 랩스 감정 결과"에 정확히 대응 — 5장 체크리스트 결과 노출 지점): 3개 탭
    - **인증 리포트**: 통과 판정 요약 문구 + 판정 기준 링크
    - **상품 이미지**: 외관 각 부위 확대 사진(카운터렁/스크러버 캐니스터/호스/밸브 등) + 부위별 상태 점(양호 등)
    - **상세 스펙**: 출시년도, 마운트타입, 제어방식, 구성품 목록, 더보기
13. **TEAMCCR 실거래가 테이블**: 거래일 · 오버홀이력(있음/없음) · 거래가 — 정렬 가능, 신품급/중고 토글, "실거래가 모두 보기"
14. **센터 정보**: 위치·운영시간·예약하기 버튼
15. **리뷰**
16. **커뮤니티 콘텐츠**(바이버 "모먼트" 대응): 실사용자가 올린 착용/사용 사진·영상 피드
17. **연관 매물**: "이런 상품은 어때요" · "동일 모델의 다른 매물"
18. **추천 매거진**
19. **신뢰 정책 아코디언**: 모든 매물에 대한 TEAMCCR 검수·인증 · 정상작동 미보증 시 환불/재정비 · 에스크로 안심결제 · 배송 중 파손 시 보상 · 센터에서 더 빠르고 안전하게
20. **판매 유도 배너**: "해당 모델을 보유하고 계신가요? 상품 판매 등록하기"
21. **법적 고지 + 사업자정보 푸터**: 6장 참고

### 판매 (독립 최상위 메뉴)

**핵심 원칙 확정**: 개인간 직거래는 운영하지 않음. **모든 매물은 TEAMCCR이 직접 접수해 상태를 검수**하고, 상태 보증·등급·상세 상태를 게시한 뒤 판매를 중개(또는 직접 매입 후 재판매)함. **검수(진단)는 판매를 위한 필수 조건**(검수 없이는 게시·판매 불가) — 단, 판매 목적이 아니어도 검수만 단독으로 신청할 수는 있음. **오버홀은 판매 시점의 선택 조건**으로, 검수 결과 필요하다고 나오거나 판매자·구매자가 희망할 때만 진행.

**메인 카드 2개**
- **위탁 판매** (바이버 "위탁 판매"에 대응, "판매속도 2배" 뱃지 방식 차용): 소유권은 판매자 유지, TEAMCCR이 접수→검수·등급판정·촬영→게시→판매 대행. 오버홀은 검수 결과에 따라 필요 시 또는 판매자/구매자 요청 시에만 별도 진행(선택)
  - 진행 순서: 판매 신청 › 장비 픽업 › **검수·등급판정 및 촬영** › 판매 시작
  - 추천 대상: 판매가 처음이라 어려운 사람 / 촬영·등록을 대행해줬으면 하는 사람 / 센터 전시를 통한 빠른 판매를 원하는 사람
  - 판매자 승인 흐름: ① 판매 신청 및 모델 선택[판매자] → ② 장비 전달(센터 방문·전문배송업체)[TEAMCCR] → ③ 검수·등급판정 및 촬영[TEAMCCR] → ④ 판매자 승인 후 판매 시작[TEAMCCR]
  - 구매 확정 후: 구매자 수령 완료 → 정산(영업일 기준 1~2일) → 정산완료
- **TEAMCCR 매입(빠른 현금화)**: 장비를 즉시 매입 → 검수 → (필요 시 오버홀) → 자사 재고로 재판매. 수익원은 매입가-판매가 마진. "가장 빠르게 현금화" 옵션으로 소구
  - 진행 순서(바이버 매입서비스 약관 기준): ① 매입 신청(입력 정보 기준 **1차 견적** 안내) → ② 동의 시 장비 전달 → ③ 검수 후 **최종 견적** 안내(1차 견적과 다를 수 있음) → ④ 판매자 동의 시 매매계약 성립 → ⑤ **3영업일 이내 등록 계좌로 대금 지급**, 이 시점에 소유권이 TEAMCCR로 이전
  - 최종 견적 거절 시 매입 중단, 반송비는 TEAMCCR 부담
  - **매입 불가 물품**: 기능 상실·심각한 손상, 심한 오염, 무단 개조·변형, 브랜드 라벨/시리얼 없음(정품 확인 불가), 비정품, 장물, 법령상 거래 금지 물품

**선택적 부가서비스**
- **오버홀**: 검수 결과 정상작동에 문제가 있거나, 판매자가 더 높은 등급으로 판매하고 싶거나, 구매자가 구매 전/후 오버홀을 요청할 경우 실비로 진행. 위탁·매입 어느 쪽이든 신청 가능
- **선정산(위탁)**(향후 검토): 위탁 판매 확정 전에 판매자에게 대금 일부를 우선 정산해주는 옵션

**배송/전달 방식** (고가 장비 특성상 바이버의 가치별 보험배송 티어를 그대로 벤치마킹):
| 장비가액 구간 | 배송비(안) |
|---|---|
| 100만원 이하 | 15,000~25,000원 |
| 500만원 이하 | 30,000~50,000원 |
| 1,500만원 이하 | 70,000~100,000원 |
| 1,500만원 초과 | 판매가 대비 요율(%) 적용 또는 센터 방문 접수만 가능 |
- 등급 구분(바이버 참고): 베이직/스탠다드/프리미엄 보험배송 — 가액이 높을수록 상위 등급만 이용 가능하도록 설계
- 센터 방문 접수 시 배송비 무료(발렛 등 부가서비스는 실비)
- 배송 중 분실·파손 시 전액 배상(보험 가입 필수)

**수수료 구조** (바이버 실제 항목 기준 재구성 — 판매자/구매자 구분):

*판매자 부담*
- 거래수수료: 위탁 판매 성사 시 일정 %(초기 프로모션 기간 무료 정책 검토)
- 주문관리수수료: 상품가의 2%(PG사 청구 수수료 + 정산대행 서비스 이용료)
- **오버홀 비용**(바이버 "업그레이드 비용"과 정확히 대응): 실비, 판매 완료 후 정산액에서 차감. 단 오버홀 완료 후 판매 포기 시에는 별도 등록 결제수단으로 청구

*구매자 부담*
- 거래수수료: 프로모션 기간 무료 검토

### 2-9. 검수 단독 서비스 가격 (바이버 "감정진단 서비스" 대응 — 판매 목적 없는 단독 신청 시)
| 유형 | 가격(안) | 범위 | 소요시간 | 장소 |
|---|---|---|---|---|
| 일반 검수 | 10~15만원 | 정품 여부, 외관 상태, 기본 작동 확인 | 30분 내외 | 센터 방문 |
| 정밀 검수 | 20~25만원 | 정품 여부 + 외관 + 호흡회로·전자계통 내부 상태 | 2~3일 | 자체 랩 |
> 매물 판매(매입형·위탁형) 진행 시에는 검수가 프로세스에 포함되어 있어 별도 비용이 아니라 거래수수료/오버홀비용 구조로 흡수됨. 위 가격은 판매 목적 없이 개인 소유 장비 상태만 확인받고 싶은 경우에 한함

**거래 취소 수수료 및 페널티** (바이버의 실제 단계별 페널티 테이블을 리브리더 거래유형에 맞게 각색):
| 단계 | 사유 | 기준가 | 페널티 비율(안) |
|---|---|---|---|
| 신청 후 | 픽업 일정 잡은 후 입고 거부 | 예상 판매금액 평균가 | 3% |
| 검수 후 | 합격 후 판매 포기 | 최적 판매가 | 3% |
| 검수 후 | 불합격(정상작동 불가, 오버홀 필요 안내 후 포기) | 검수비 실비 | 정액(예: 30만원) |
| 판매 중 | 판매 포기 | 최적 판매가 | 3% |
- 초기 오픈 유예기간(예: 오픈 후 6개월)은 페널티 상한액을 낮게 제한해 진입장벽을 낮추는 방식 검토(바이버는 최대 200만원 상한 운영)
- 상세 내용은 마이페이지 하위 "거래 취소 수수료 및 페널티 정책" 페이지로 안내

**참고 지표**(마케팅 문구 아이디어): 바이버는 "직접판매 평균 13.2일 vs 위탁판매 평균 5.5일"처럼 위탁의 속도 이점을 수치로 보여줌 — TEAMCCR도 위탁 판매의 평균 판매 소요기간을 수치로 노출하면 신뢰도 확보에 효과적

진행 흐름 요약: (위탁 판매) 신청 폼(브랜드/모델/구매연도/누적 다이빙 횟수/오버홀 이력/사고·침수 이력) → 진행 트래킹(접수→입고→검수·등급판정→(선택)오버홀→게시→판매완료→정산) / (매입형) 신청 후 즉시 매입가 산정·현금화

### TEAMCCR (팀 소개 — GNB 최상위 유지)
실제 teamccr.net 메뉴 구조를 참고하되 아래처럼 축소·정리:
- **센터 소개**: "다양한 형태의 다이빙을 재호흡기 다이빙을 중심으로 더 안전하고 재미있게 즐길 수 있도록 돕는다"는 미션 소개 + "TEAMCCR IN KOREA" — 재호흡기 교육, 텍다이빙 과정, 강사·트레이너 과정 운영, 인근 수영장 연계 장비 테스트/원포인트 레슨 안내
- **강사진**: 책임 트레이너 프로필(12장 참고) — 자격, 브랜드별 트레이너 인증, 연락처(이메일/전화/카카오톡). 운영진 소개를 별도로 두지 않고 이 섹션으로 통합(운영진 = 강사진과 동일 인물)
- **찾아오시는 길**: 위치, 오시는 방법
- **팀 공식 스케줄 캘린더**: 팀 다이빙 일정(정기 다이빙, 원정, 교육 일정, 이벤트 등)을 월간 캘린더 형태로 공개. 클릭 시 일정 상세(장소, 참가 트레이너, 신청 가능 여부) 노출

### 매거진 (GNB 최상위 유지 — 바이버와 동일)

### 마이페이지
- **로그인 전 화면**: 로그인/회원가입 유도 배너 + 프로모션 캐러셀 + 정책·고지 메뉴 리스트(이벤트 · 공지사항 · 고객센터 · 이용정책 · 브랜드 소개)

- **이벤트 페이지**: 배너 카드 목록 형태(진행 중 이벤트 + "지난 이벤트 보기"). TEAMCCR 예시: 매입 프로모션, 오버홀 할인 이벤트, 해외 배송 안내, **TEAMCCR 검수 서비스 소개**(바이버 "전문 엔지니어의 진단·VIVER VERIFY"에 대응), 위탁 판매 빠른 정산 홍보

  - 자주 묻는 질문: 검색창 + 카테고리 필터(전체/공통/해외/판매/매입/구매) — 바이버는 "공통·해외·판매·매입&보상·구매" 5개 카테고리 운영, TEAMCCR도 동일 구조 채택
  - 1:1 문의하기: 운영시간 명시(예: 평일 11:00~18:00, 주말/공휴일은 센터 방문 관련 문의만) + 1:1 상담·전화 상담 버튼

- **이용정책**: 바이버 확인 결과 **19개 항목이 별도 페이지가 아니라 하나의 "이용정책" 문서 안에 번호별 섹션으로 통합**돼 있고, 검수 화면·거래취소 화면 등 다른 곳에서 "진단 정책"·"거래취소 수수료 및 페널티 정책" 등을 누르면 이 문서의 해당 섹션으로 앵커 링크만 이동함. TEAMCCR도 이 구조 그대로 채택 — 19개 별도 페이지 대신 **단일 이용정책 문서 + 섹션 앵커(#3-검수정책, #9-거래취소수수료 등)**로 구현하면 관리 부담이 훨씬 적음. 목록:
  1. 구매/판매 프로세스 및 관련 정책
  2. 상품 가격 정책
  3. **검수 정책**(바이버 "진단 정책"에 대응)
  4. **오버홀 정책**(바이버 "업그레이드 정책"에 대응 — 선택적 부가서비스 규정)

  6. 배송 정책
  7. **워런티 정책** (도입 여부 신중 검토 — 바이버는 이 정책을 2023년 도입했다가 2025.08.06부로 중단함. 운영 부담이 컸던 것으로 추정되므로, TEAMCCR은 오버홀 보증기간 정책(3장)으로 대체하고 별도의 장기 무상수리 워런티는 초기엔 두지 않는 방향 권장)
  8. 부적절 행위 금지와 이용 제한 정책
  9. 판매 취소 수수료 및 페널티 정책
  10. 이상 거래 상품의 거래 취소 정책
  11. 회원 가입/계정 관리/탈퇴 정책
  12. 거래 수수료 및 비용 정책
  13. 가품·오작동 보상 정책
  14. 포인트/적립 정책(도입 시)
  15. TEAMCCR 매입 상품 정책
  16. 사업자 및 세무 관련 정책
  17. 교육 과정 관련 정책
  18. 커뮤니티 콘텐츠 정책(바이버 "Moment 서비스 정책"에 대응)

  > **검수(진단)는 판매를 위한 필수 조건**(생략 불가) — 단, 판매 목적이 아닌 개인 소유 장비의 상태 확인용으로 검수만 단독 신청하는 것은 가능. **오버홀은 판매 시점의 선택 조건**으로, 검수 결과 필요 판정이 나오거나 판매자·구매자가 희망할 때만 진행

- **이용약관(별도 문서, 이용정책과 다름)**: 바이버 확인 결과 "이용약관"은 "이용정책"과 별개의 공식 계약 문서 — 조문 형태(제1조~제N조)로 정의·계약성립·해지·통신판매중개 부인·검수·청약철회 제한·수수료·매입서비스·Moment서비스·면책·관할법원 등 법적 근거를 규정하고, 세부 운영 기준(가격표, 절차 등)은 "이용정책"을 인용하는 구조. TEAMCCR도 **이용약관(법적 계약) + 이용정책(운영 세부사항) 이원 구조**로 준비 필요(변호사 검토 필수)
  - 이용약관 핵심 조항 체크리스트: 용어 정의(회원/판매자/구매자/검수/검수착오상품/배송업체 등), 이용계약 체결·해지 요건(만 14세 미만 가입 불가 등 — 다이빙 특성상 자격증 연령 기준과 별도 검토 필요), 통신판매중개 및 대리행위 부인(위탁형), 보증의 부인(검수실책 제외), 청약철회 제한(위탁형), 서비스수수료·페널티 부과 방식, 매입서비스 조항, 커뮤니티 콘텐츠 조항, 면책조항, 관할법원(해외 거주자는 국내 법원 관할로 명시 검토)
- **바이버 원 방식의 AI 어시스턴트**(향후 검토, 신규 아이디어): 바이버는 "바이버 원"이라는 자체 AI 챗봇을 운영 — 상품/시세 정보 검색, 구매입찰 등 주요 기능 안내, 이용내역 조회·요약 지원. TEAMCCR도 향후 "장비 추천/오버홀 상태 조회/검수 결과 설명"을 도와주는 AI 어시스턴트 도입을 3단계 로드맵 이후 검토 가능

- **설정 페이지**(마이페이지 하위, 정책 리스트와는 별도): 계정 관리(로그인/회원가입) · 언어 변경 · 이용약관 · 이용정책 · 오픈소스 라이선스 · 개인정보 처리방침 · 버전정보

### 2-2. 상태등급 체계 (바이버의 미착용/중고/빈티지 구분을 리브리더 맥락으로 재설계)

| S(신품급) | 미개봉 또는 실사용 이력 없음, 오버홀 이력 없이도 전 항목 정상 |
| A(정상) | 실사용 이력 있으나 외관·호흡회로·전자계통 전 항목 정상, 오버홀 불필요 |
| B(사용감 있으나 정상작동) | 외관 마모·스크래치 등 경미한 사용감 있으나 기능상 정상, 오버홀 선택 가능 |
| C(오버홀 필요) | 검수 결과 호흡회로·전자계통 중 일부가 기준 미달 — 오버홀 진행해야 판매 가능(바이버 "조건부 합격"에 대응) |
| 불합격 | 외형 변조, 전체/부분 비정품 부품, 오버홀로도 상품화 불가, 신청 정보와 실물 불일치 등 — 판매 불가 |

### 2-3. 검수 항목 (바이버 "외부 컨디션/내부 성능" 2단 구조를 그대로 채택)
- **외관 검수**: 광택·변색·찍힘·눌림·스크래치 여부 — 프레임, 카운터렁, 하네스, 마스크/DSV, 호스 외피
- **기능 검수**: 음압/양압 누기, 산소센서 정확도, 솔레노이드·배터리 상태, 스크러버 상태 — 오버홀/수리 필요 여부 최종 판정

### 2-4. 거래 진행 규칙 (바이버 원문 반영)
- **결제 우선 원칙**: 거래 체결은 '신청' 순서가 아니라 '결제 완료' 순서로 확정 — 동시에 여러 명이 관심을 보여도 결제를 먼저 완료한 구매자에게 거래가 성립
- **이상 거래가 정책**: 시세 대비 비정상적으로 낮거나 높은 가격은 사전 고지 없이 판매 중단·거래 취소될 수 있음(가격 조작·자전거래 방지 목적)
- **위탁 판매 접수 마감**: 픽업 일정이 3영업일(주말·공휴일 제외) 내 확정되지 않으면 판매 포기로 간주, 거래 취소 수수료 부과
- **검수 중에는 판매 포기 불가**: 접수~검수 완료 전까지는 판매 포기 자체가 불가, 검수 완료 후에만 포기 가능(단 페널티 발생 가능)

### 2-5. 판매 포기·거래 취소 수수료 (바이버 실제 비율 기준 각색)
| 단계 | 위탁형 | 매입형 |
|---|---|---|
| 픽업 확정 전 단순 변심 | 예상 판매금액 평균가의 1.8% | 해당 없음(즉시 매입이므로 신청 철회만 가능) |
| 픽업 완료~검수 중 | 포기 불가 | - |
| 검수 결과 '정상' 안내 후 포기 | 최적판매가의 1.8% | - |
| 검수 결과 '오버홀 필요' 안내 후 포기 | 페널티 없음(오버홀 거부는 정당한 포기 사유로 인정, 반송 처리) | - |
| 검수 결과 '불합격' — 비정품 의심 | 예상 판매금액 평균가의 10% | - |
| 검수 결과 '불합격' — 상태 불일치 등 | 예상 판매금액 평균가의 1.8% | - |
| 판매 게시 중 포기 | 최적판매가의 1.8% | - |
> 오버홀(업그레이드) 완료 후 판매 포기 시에는 오버홀 비용을 별도 등록된 결제수단으로 청구
> 페널티 감면 사유(소명 가능): 배송업체 귀책, 센터 직접 방문으로 배송비 미발생, 접수 후 90일 경과 후 포기, 비정품 의심 건에 고의성 없었음을 객관적으로 소명 가능한 경우

### 2-7. 반품 정책 — 거래유형별 적용 여부 (중요, 바이버 원문 기준 확정)
- **매입형(TEAMCCR 직접판매)**: TEAMCCR이 판매자이므로 **전자상거래법상 청약철회(7일 이내 반품) 적용 대상**. 단, 구매자 단순변심 반품 시 반품 배송비는 구매자 부담
- **위탁 판매**: TEAMCCR이 통신판매중개자이므로 **청약철회(단순변심 반품)는 적용되지 않음** — 결제 완료 후에는 원칙적으로 취소·반품 불가(단일 재고 상품 특성)
- **단, "검수 착오 상품" 환불은 별개 구제수단으로 존재**(바이버 "진단 착오 상품" 개념): TEAMCCR의 검수 기준에 미달함에도 검수를 통과시켜 판매된 경우, 구매자는 **수령일로부터 7일 이내 환불(또는 A/S 대체) 가능** — 이는 청약철회가 아니라 TEAMCCR 자체 검수 실책에 대한 책임이므로 위탁형에도 예외적으로 적용
- 반품 불가 사유(매입형에도 적용): 포장 개봉으로 가치 훼손, 인증서/라벨 분실, 사용으로 인한 가치 감소, 구성품 분실·파손

### 2-8. 가격 정책 (바이버의 3단계 가격 개념을 그대로 채택)
| 개념 | 정의 |
|---|---|
| **TEAMCCR 인덱스** | 브랜드·모델별 최근 거래 시세 참고 자료(빅데이터 기반) — 상태등급·구성품 포함 여부는 반영되지 않은 원시 시세 |
| **예상 판매금액** | 판매 신청 시 입력 정보 + 인덱스 기반으로 미리 안내하는 대략적 가격 범위(검수 전) |
| **최적 판매가** | 검수 완료 후 실제 상태·구성품을 반영해 제안하는 확정 가격 — 판매자가 자유롭게 변경 가능 |
> 해외 판매자에게는 해외결제대행 수수료·환변동 비용을 반영한 별도의 "해외노출가격"(외화 환산)을 적용하는 것도 검토(10장 다국어·해외 정책과 연계)

## 5. 검수·인증 체크리스트

TEAMCCR에 접수되는 모든 매물(매입형·위탁형 공통)은 오버홀 진행 여부와 무관하게 아래 체크리스트로 **검수·등급판정**을 받고, 그 결과가 상세페이지에 상태 보증·등급·상세 상태로 게시됨. 검수 결과 오버홀이 필요하다고 판단되거나 판매자·구매자가 원할 경우에만 별도로 오버홀 진행(3장 참고):

- 외관: 프레임, 카운터렁, 하네스 마모도
- 호흡회로: 음압/양압 누기테스트
- 전자계통: 산소센서 밀리볼트값·잔여수명, 솔레노이드, 배터리, HUD/컨트롤러
- 스크러버: 캐니스터 상태, 소다라임 종류 및 잔여 사용가능 시간
- 호스/1·2단 레귤레이터: 누기, 호흡저항 테스트
- 이력 검증: 시리얼 기반 오버홀 이력 대조
- 검수 결과 발급 포맷: PDF 인증서 + QR코드 온라인 조회(위변조 방지)

## 6. 법적/신뢰 요소

**거래유형별 법적 지위 재정리** (바이버 실제 상품페이지 법적 고지 문구 확인 결과 반영):

| 거래유형 | TEAMCCR 법적 지위 | 책임 범위 |
|---|---|---|
| 매입형(직접매입 후 재판매) | **통신판매업자**(거래 당사자) — 소유권을 넘겨받아 직접 판매 | 상품 자체에 대한 책임 전반 |
| 위탁 판매 | **통신판매중개자** — 소유권은 판매자 유지, TEAMCCR은 중개자 | 거래 자체(하자·청약철회 등)는 판매자 책임. **단, TEAMCCR이 제공하는 검수·인증 서비스 자체의 품질에 대한 책임은 TEAMCCR에 있음** |

> 바이버 상품페이지 실제 고지 문구 예시(참고용, 그대로 복제 금지): "(주)OOO는 통신판매중개자이며 통신판매의 당사자가 아닙니다. 본 상품은 개별 판매자가 등록한 상품으로 상품, 상품 정보, 거래에 관한 의무와 책임은 해당 판매자에게 있습니다. 단, 거래 과정에서 제공하는 진단서비스에 대한 책임은 (주)OOO에 있습니다." → TEAMCCR 버전으로 각색해 **위탁형** 상품 페이지 하단에 노출

- 통신판매업 신고
- 전자상거래법 청약철회 규정 — 매입형(직접 책임)은 안전장비 특성상 예외조항 법률 자문 필요, 위탁형은 통신판매중개자 면책 범위 내에서 정리(단 검수·인증 서비스 품질 책임은 별도 존재)
- 검수·오버홀 작업 자체의 하자(판정 실수 등)에 대한 책임 범위는 거래유형과 무관하게 TEAMCCR에 있음 — 약관에 명시(변호사 검토 필수)
- 개인정보처리방침 / 이용약관
- **본인인증**: 국내 회원은 휴대폰 본인인증(고가 거래·안전장비 특성상 권장), 해외 회원은 한국 휴대폰 번호가 없으므로 본인인증 절차를 면제하고 이메일 인증 + 소셜 로그인(Google/Apple)으로 대체 — 국가별 가입 절차 이원화 필요
- 고가 거래 특성상 에스크로 결제 검토
- 오버홀 작업 배상책임보험 가입 검토
- **사업자정보 푸터**(바이버 하단 고지 형식 참고): 상호·대표자명·사업자등록번호·통신판매번호·사업장 주소·센터(쇼룸) 주소·이메일·호스팅 서비스 — 모든 페이지 하단 공통 노출

## 7. 콘텐츠/자산 준비

- 브랜드별 제품 스펙 DB(모델명/출시연도/정가/스펙/매뉴얼 링크)

- 브랜드/딜러 파트너십 소개 페이지
- **개인정보 처리방침**(바이버 원문 구조 참고): 처리목적(회원가입·서비스제공·마케팅·이벤트) / 보유기간(회원탈퇴시까지, 법정보관 5년·3년·3개월 구분표) / 수집항목(가입유형별·거래유형별 구분) / 제3자 제공 원칙(미제공, 예외사유 명시) / 처리위탁 업체 목록(PG사, 본인인증사, 알림톡, CS, 배송업체, 클라우드, 분석툴 — 국내/국외 위탁 구분) / 파기절차 / 정보주체 권리행사방법 / 쿠키·행태정보 수집고지 / 개인정보보호책임자 연락처 / 아동(14세 미만) 가입 제한 고지

### 이미지·미디어 리소스 체크리스트
- **TEAMCCR 브랜드**: 로고(SVG, 라이트/다크), 파비콘, 앱 아이콘(1024×1024), 인증마크 씰 그래픽
- **취급 장비 브랜드 로고**: 11장 DB 기준 브랜드별 공식 로고 — 딜러 계약서에 로고 사용 승인 조항 확인 필요
- **매물 사진**: 브랜드/모델별 대표 이미지, 인증 체크리스트용 부위별 클로즈업(카운터렁·스크러버 캐니스터·호스·밸브·산소센서), 가능하면 360° 촬영 세트
- **사람·공간 사진**: 강사진 프로필(박정훈·백승균, 통일된 톤), 센터/작업장 사진, 팀 활동 사진(매거진·커뮤니티용)

## 8. 기술/기능

- 회원 시스템: 개인/딜러/교육생 구분, 등급별 권한
- **회원 탈퇴/휴면 정책**(바이버 원문 기준): 거래 진행 중·페널티 미납·이용정지 상태에서는 탈퇴 제한, 탈퇴 시 개인정보 즉시 삭제하되 전자상거래법(계약·결제기록 5년, 분쟁처리기록 3년)·통신비밀보호법(접속기록 3개월)에 따른 법정 보관기간 준수, 탈퇴 후 15일간 재가입 제한, 1년 미접속 시 휴면 전환(30일 전 사전 고지)
- **로그인/가입**: 이메일+비밀번호, 소셜 로그인(카카오·Apple·Google), 이메일 인증코드 발송/확인 플로우, 국내(휴대폰 본인인증)·해외(간편가입) 이원화 가입 로직
- 매물 관리 백오피스: 접수→검수·등급판정→(선택)오버홀→가격책정(매입형)/정산(위탁형)→게시→판매→정산
- 예약 시스템: 오버홀·교육 캘린더/슬롯
- 팀 스케줄 캘린더: TEAMCCR 페이지에 노출되는 공개용 월간 캘린더(읽기 전용) — 예약 캘린더와 별도 관리, 관리자가 일정 등록/수정
- 시세 정보: 모델별 **실거래가 테이블**(거래일·오버홀이력·거래가, 정렬 가능, 신품급/중고 토글) — 옵션이 아니라 상품 상세페이지 필수 요소로 격상(바이버 확인 결과)
- 검색/필터: 브랜드, 모델, 마운트타입, 제어방식(mCCR/eCCR/Hybrid/SCR), 가격, 사용시간/다이빙횟수, 구매·제조연도, 판매방식(매입형/위탁형), 상태등급, 지역
- 이미지: 360° 뷰 옵션(제품 촬영 시 360도 회전 이미지 세트 필요)
- 인증서 조회: 시리얼/QR 기반
- 커뮤니티 콘텐츠(바이버 "모먼트" 대응): 사용자가 올리는 착용·사용 사진/영상 피드, 좋아요 기능 — 매물 상세페이지에 노출. 게시 조건(모델 태그 필수, 시리얼/QR 노출 금지, 광고성·비방 콘텐츠 금지), 신고 3회 누적 시 자동 숨김, 계정 제한(30일)까지 바이버 정책 구조 준용
- **TEAMCCR CARE 멤버십**: 구매 시점 업셀 배너로 노출, 매입형(직접구매) 상품 구매자 대상. 가입 후 일정 기간(예: 2~3년) 무상 점검·정기 오버홀 횟수 제한 없이 제공. 가격은 장비가액에 따라 차등(바이버케어 40만~190만원 티어 참고해 리브리더 가격대에 맞게 재산정, 예: 500만원 미만 장비 15만원 / 500만~1500만원 25만원 / 1500만원 이상 40만원 등 초안). 판매 시 구매자에게 자동 양도, 미사용 상태로 만료 시 가입비 일부 포인트 환급 검토, 가입 후 7일 이내 미사용 해지 시 전액 환불

- 알림: 오버홀 주기, 관심매물 가격변동, 신규 입고
- 결제/정산: 위탁판매 자동정산, 에스크로 연동
- **포인트/적립 정책**(향후 검토, 바이버 참고): 구매 결제액의 0.3% 적립, 리뷰 작성 시 정액 적립, 1년 미사용 시 자동 소멸, 현금 전환 불가, 해외 회원은 적립 제외 검토
- **가격 제안(입찰) 기능**(향후 검토): 구매자가 게시 가격보다 낮은 희망가를 제안하면 판매자가 확인 후 즉시 수락(즉시 거래 체결) 가능한 기능 — 위탁 판매 매물의 재고 회전 촉진 목적, 해외 구매자는 이용 제한 검토

## 9. 플랫폼 전략 (데스크톱 · 모바일웹 · 앱)

**구조**: 하나의 API + 여러 프론트엔드. 단, 고객 화면과 관리자(백오피스) 화면은 레이아웃 전략을 분리.

| 구분 | 기술 방향 |
|---|---|
| 고객 화면 — 데스크톱 웹 | **바이버 확인 결과 채택**: 별도 데스크톱 그리드를 만들지 않고, 모바일 앱 화면을 폰 너비 컬럼 그대로 중앙(또는 우측)에 띄움. 나머지 여백은 QR코드+태그라인으로 "모바일 앱으로 이용하기" 유도 영역으로 활용 |
| 고객 화면 — 모바일 웹 | 반응형 웹 (동일 코드베이스, 1열 레이아웃) — 데스크톱과 완전히 동일한 화면을 그대로 사용 |
| 고객 화면 — 앱 | 웹사이트를 그대로 앱으로 포장(Capacitor 방식) — 화면을 새로 만들지 않고 지금 만드는 웹사이트를 그대로 앱스토어·플레이스토어용 앱으로 전환 |
| 관리자(백오피스) | **데스크톱 우선** 설계 — 벤치마킹 대상이 아닌 내부 전용 툴(외부에서 확인 불가능), 매물 관리·오버홀 진행상황·정산 등 표·필터·대시보드 중심 업무 특성상 모바일 대응은 최소한만 |

> **정정 사항**: 이전 버전에서는 "데스크톱은 넓은 화면에 맞는 별도 그리드로 확장"을 권장했으나, 실제 viver.co.kr 데스크톱 화면을 확인한 결과 **바이버 본인도 별도 그리드 없이 모바일 앱 폭 그대로 노출**하고 있음을 확인. 하나의 모바일 우선 화면만 만들면 되므로 오히려 개발 리소스가 더 절약되고, "바이버와 완전히 동일하게" 원칙에도 부합해 이 방식으로 전환.
> 데스크톱 여백 영역 활용안: 브랜드 태그라인 + 앱 다운로드 QR코드(바이버와 동일 패턴), 필요 시 프로모션 배너 등 추가 검토
> **앱 전환 방식 확정**: 웹사이트를 새로 개발하지 않고, 완성된 웹사이트를 그대로 앱 형태로 감싸서(Capacitor) 앱스토어에 올림. 화면·기능은 웹과 100% 동일, 개발은 한 번만 하면 됨.
> 관리자 화면은 바이버 사례와 무관하게 기존 방침(데스크톱 우선) 유지 — 공개된 정보가 없어 대조 자체가 불가능하고, 내부 업무 도구는 일반적인 SaaS 대시보드 설계 원칙을 따르는 것이 타당

**설계 원칙**
- API-first: 웹/앱 공용 REST·GraphQL API
- 인증: JWT 등 토큰 기반(앱 전환 대비)
- 이미지·PDF: CDN 연동
- 결제: 웹·앱 공통 지원 PG SDK 확인
- 알림: 이메일/문자(웹) · 푸시(앱) 채널 추상화 설계

## 10. 다국어 지원 (한국어 · 영어 · 중국어 · 일본어)

**기본 언어는 한국어**. 사업 기반이 국내 센터·강사진에 있고 주 이용자도 국내 다이버이므로 한국어가 기본값이며, 해외 다이버에게도 서비스를 확장하고 싶어서 언어 전환 시스템을 별도로 두는 구조 — 영어가 기준이 아니라 한국어가 기준.

### 고정 표기 원칙 (번역하지 않음)
아래 항목은 **언어 설정과 무관하게 항상 영문 그대로 고정** 노출:
- 브랜드·모델명 (TEAMCCR, AP Diving, Liberty, mCCR 등)
- 브랜드 모토 ("Keep silent without bubble.")
- GNB 메뉴명 (Home, Used Gear, New Gear, Overhaul & Service, Education, Inspection & Diagnostics, Magazine, TEAMCCR, Support, My Page 등 확정된 영문 메뉴 라벨)

### 번역 대상 (언어별 로컬라이즈)
위 고정 항목을 제외한 **설명형 콘텐츠**(히어로/섹션 리드 카피, 상품 상세 설명, 안내문, 매거진 본문, 강사 소개 문단, 고객센터 안내 등)는 4개 언어로 번역.

### 자동감지 & 전환
- `navigator.language`(Accept-Language) 우선 → 지원 언어(ko/en/zh/ja)와 일치하면 해당 언어, **미지원 언어는 한국어로 폴백**(기본 언어가 한국어이므로 영어가 아닌 한국어 기준). 수동 전환 시 쿠키+계정에 저장
- URL 구조: `/ko/`(기본) `/en/` `/zh/` `/ja/` + hreflang 태그 + 다국어 sitemap
- 전환 UI: GNB 우측, 국기 아이콘 대신 텍스트 라벨("한국어/EN/中文/日本語"), 한국어가 첫 번째 항목

### 구현 방향
- 번역 대상 UI 문자열은 i18n 라이브러리(next-intl, react-i18next 등) JSON 리소스로 관리, 고정 표기 항목은 번역 키 자체를 만들지 않고 상수로 하드코딩해 오역 방지
- 동적 콘텐츠(매물 설명, 매거진 등)는 DB에 언어별 필드(`desc_ko`, `desc_en`...) 또는 별도 번역 테이블로 저장

### 번역 범위 우선순위
| 범위 | 처리 |
|---|---|
| 고정 표기(브랜드·모토·메뉴명) | 항상 영문, 번역하지 않음 |
| 매물 상세·안내문·안전고지 | 필수, 전문 번역가 검수 |
| 검수 인증서 | 한국어+영어 우선, 중/일 2단계 |
| 매거진 | 우선순위 낮음, 순차 번역 |
| 강사 프로필 소개 문단 | 4개 언어 |

### 부가 고려사항
- **통화는 언어와 독립된 별도 설정**(바이버 확인 결과: 지역/언어/통화 3개를 각각 선택) — 언어를 영어로 바꿔도 통화는 자동으로 바뀌지 않고, 사용자가 별도로 KRW/USD 등을 선택. 가격은 원화 기준 + 참고 환산통화 병기 검토, 수심 단위는 미터(m) 고정
- **우선순위**: 1단계 한국어+영어 동시 오픈 → 2단계 중국어·일본어 확장
